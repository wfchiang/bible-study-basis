# **保罗交付提摩太与提多的任务**

保罗差派提摩太去以弗所以及差派提多去克里特的时间点并没有明确记载在圣经里，很可能都是在保罗第一次被囚禁获释以后的布道旅行中。提摩太前书与提多书写成的时间也大约是这个时期，而提摩太后书是保罗第二次被囚禁，临近殉道的时候。

从提摩太前后书以及提多书来看，保罗差派提摩太与提多都是任务性的。他期望这两人处理完教会的事务以后就来与他会合。他们任务的大方向是相同的，但是根据教会的背景，他们具体处理的问题并不完全相同。

| 提摩太 (去以弗所教会) | 提多 (去克里特教会) |
| ----- | ----- |
| 圣洁生活，树立生命典范 坚守真理，传讲纯正福音 设立合适的领袖 解决教会面临的问题 |  |
| **历史：**以弗所教会是保罗第三次宣道旅行时建立 (徒19章)。当提摩太到达时，教会已经有长老们管理了好一段时间有。 | **历史：**克里特岛可能是保罗第一次被囚获释以后造访。保罗可能传了福音，草创教会以后便留下提多，自己往下一个目的地去。 |
| **假师傅与教会领导的问题：**这个问题在保罗离开以弗所教会时已经预言： “必有凶暴的豺狼进入你们中间，不爱惜羊群” (徒20:29b)。 | **缺乏领袖以及会众没有生命见证的问题：**当时克里特岛可能长期经受海盗侵袭，社会秩序混乱，居民道德沦丧。 |

提摩太前书与提多书的内容都是保罗对于任务的教导。从结果来看，提多顺利地完成了保罗交代的任务，而提摩太却遇到了麻烦。在保罗临近殉道，写下提摩太后书时，提摩太仍在以弗所，而提多已经完成了在克里特的任务，被差遣到另一个地方去了 (达马太，提后4:10)。从提摩太书来看，提摩太当时似乎因为所遇见的困难而感到沮丧。有人认为提摩太是因为个性比较懦弱，而提多比较坚强，所以提摩太在服事上不顺利，还需要保罗些提摩太后书鼓励他。这的确是一种可能。但有没有另一种可能是：保罗交代给提摩太的任务比提多的要难得多? (处理教会领导层的问题比建造会众要难得多吗？)

总的来说，提摩太前书与提多书可以看成是保罗给提摩太与提多的 “任务提示”。这两本书在相近的时间写成，主要针对他们完成任务的具体意见。而提摩太后书是保罗写给提摩太的一封勉励信。

# **给在苦难中神仆人的激励**

提摩太后书主要是保罗写给提摩太的一封勉励信。提摩太在以弗所服事了好一段时间，却似乎没有成效，无法完成保罗交代给他的任务 (其实是神国里的任务)。我姑且用三段经文总结保罗的勉励：

* ***神赐给我们**，不是胆怯的心，乃是刚强、仁爱、谨守的心。 \[提后1:7\]*  
* *你不要以给我们的主作见证为耻，也不要以我这为主被囚的为耻；总要按神的能力，与我**为福音同受苦难**。 \[提后1:8\]*  
* *那美好的仗我已经打过了，当跑的路我已经跑尽了，所信的道我已经守住了。从此以后，有公义的冠冕为我存留，就是按着公义审判的主到了**那日要赐给我的**  \[提后4:7-8a\]*

\[提1:7\] 告诉我们，神已经先给我们服事的恩赐，就是 “*刚强，仁爱，谨守的心*”。服事不是神压榨我们，好像从前埃及人虐待以色列人，要以色列人做出砖头却不给材料。我们能服事终究是得了神的赏赐在作神的仆人。不是我们加添什么给神，而是神赏赐我们。我们也应该相信祂所给的恩赐也一定足够克服所面临的困难。

\[提1:8\] 告诉我们，要与保罗一同为福音同受苦难。这话虽然是对提摩太说的，但这话也是圣灵为我们保留在圣经里的。所以，我们不要说 “是提摩太要受苦难”，而要知道是我们都要与保罗同受苦难。圣经也明说了：

*从这一切苦难中，主都把我救出来了。不但如此，**凡立志在基督耶稣里敬虔度日的也都要受逼迫。** \[提后3:11b-12\]* 

也不要说 “这是断章取义，其实服事不一定要受苦”，保罗一再强调服事一定会受苦：

*你要和我同受苦难，好像基督耶稣的精兵。 \[提后2:3\]*

*你却要凡事谨慎，忍受苦难，做传道的工夫，尽你的职分。 \[提后4:5\]*

\[提4:6b-8a\] 告诉我们，在世上为神为福音的缘故受苦不是枉然，神必定要奖赏我们。这是我们服事的盼望。因为期盼在天上的奖赏而服事，这种心态会很功利主义吗？“为了神的赏赐” 是一个合适的服事动机吗？问题是，你到底相信不信赏赐是神将要做的事？ 圣经说：“*不但赐给我 (保罗)，也赐给凡爱慕他显现的人 \[提后4:8b\]*”；你期待神完成祂的这个应许吗？

# **神的仆人的使命**

尽管以弗所教会与克里特岛在风土民情，教会规模，会众的灵命情景，等方面有很大差异，但是保罗交付给提摩太与提多的任务却有很多共同点；任务的大方向是相同的。

## **圣洁生活，树立生命典范**

保罗向提摩太与提多都强调：神的仆人**自己**必须圣洁。

*在大户人家，不但有金器银器，也有木器瓦器；有作为贵重的，有作为卑贱的。人若**自洁**，脱离卑贱的事，就必作**贵重的器皿**，**成为圣洁，合乎主用**，预备行各样的善事。 \[提后2:20-21\]*

*劝 (克里特的) 年轻的男子要自律。无论在甚么事上**你都要显出好行为的榜样**，在教导上要纯全，要庄重，言词要纯正，无可指摘，使反对的人因为无从毁谤，就自觉惭愧。 \[多2:6-8，新译本\]*

要服事神，自己的圣洁生活是很重要的；重要的是作**贵重的器皿**，因为只有贵重的器皿会被主人爱惜。我们在教会中参与服事可以看作是神的器皿，然而这些器皿不全然都是贵重的。谁是贵重的器皿呢？不是付出时间多的人，也不是领导的人，而是自洁，脱离卑贱的事的人作贵重的器皿。

打个比方，我们常说 “我们服事是献给神的”。如果你的服事是一块献给神的牛排，你的生命见证就是盘子。无论你把牛排煎得多完美，如果盘子是脏的，牛排就是脏的。

## **讨神喜悦，坚守真理，传讲纯正福音**

以弗所教会和克里特教会同样遭遇的问题就是教会当中有人抵挡真理。在以弗所教会有假教师宣讲 “已经不再有复活” 这种与真理完全不符的教训 (提后2:17-18)。在克里特教会有人喜欢教导些无意义的事并导致纷争 (提3:9)。然而保罗预言了 (主耶稣先预言了[^1]) 将来这种抵挡真理的人在教会当中将会比比皆是：

*末世必有危险的日子来到。因为那时，人要专顾自己，贪爱钱财，自夸，狂傲，...，**有敬虔的外貌，却背了敬虔的实意**；这等人你要躲开。 \[提后3:1b-2a,5a\]*

*因为时候要到，人必厌烦纯正的道理，耳朵发痒，就随从自己的情欲，增添好些师傅，并且掩耳不听真道，偏向荒渺的言语。 \[提后4:3-4\]*

坚守真理并且传讲真理听起来容易，其实是困难的。因为这些抵挡真理的人有敬虔的外貌，就在你身边，但是他们不喜欢真理，反而会创造一堆似是而非的道理当作真理来教导别人。当神的仆人传讲真理的时候，会被这些人鄙视，(我不是说被鄙视的都是神的仆人)，而当神的仆人顺着他们去讲那些似是而非的道理时，会得到赞美，(我也不是说被赞美的都是被迷惑的)。就如圣经所说：

*人都说你们好的时候，你们就有祸了！因为他们的祖宗待假先知也是这样。 \[路6:26\]*

*我现在是要得人的心呢？还是要得神的心呢？我岂是讨人的喜欢吗？若仍旧讨人的喜欢，我就不是基督的仆人了。 \[加1:10\]*

面对这样的挑战，保罗要提摩太先谨记自己的身份是神的仆人，要讨神的喜悦。这不只是神仆人的一个心态，也是一切服事的动机，更是抵挡那些有敬虔外貌的假教师的秘诀。

*你要和我同受苦难，好像基督耶稣的精兵。凡在军中当兵的，不将世务缠身，好叫**那招他当兵的人喜悦**。 \[提后2:3-4\]*

*你当竭力**在上帝面前得蒙喜悦**，作无愧的工人，按着正意分解真理的道。 \[提后2:15\]*

再者是要坚守真理。他们自己必须先熟悉神的话语，才能保护自己不受异端的攻击，也不陷入虚无的争辩。

*作恶的和迷惑人的，必越久越恶，他欺哄人，也被人欺哄。**但你所学习的，所确信的，要存在心里**；因为你知道是跟谁学的，并且知道你是从小明白**圣经** (scripture)，这圣经能使你因信基督耶稣，有得救的智慧。 \[提后3:13-15\]*

把真理存在心里更进一步就是帮助那些已经背离真理，听信错误教导或是沦陷在无意义的争辩中的弟兄姐妹带回到正道上：

*圣经都是神所默示的，于教训、督责、使人归正、教导人学义都是有益的，叫属神的人得以完全，预备行各样的善事。 \[提后3:16-17\]*

*你当竭力**在上帝面前得蒙喜悦**，作无愧的工人，**按着正意分解真理的道**。 \[提后2:15\]*

## **传承 \-- 设立合适的领袖**

保罗在提摩太前后书与提多书中都教导他们选取教会领袖的条件。当然，在如今的教会中，不是每一个人都有机会考核牧师，长老，或执事等教会领袖。但广义的来说，如果你热心投入一项事工，那么总有一天你要把它传承给下一代。到时你该交给怎样的人呢？我想无论是不是作领袖，秉持选取领袖的原则来传承都是合适的。

保罗给提摩太和提多他们个人有两个重要的嘱咐：(1) 圣洁生活，树立典范，(2) 讨神喜悦，坚守真理，传讲纯正真理。然后，保罗要他们去为教会设立合适的领袖。这些领袖要符合什么条件呢？还是一样的！就是圣洁，活出见证，并讨神喜悦，坚守并传讲真理。当然，保罗也不吝列出具体的细项可供他们使用：

*照我所吩咐你的，在各城设立长老。若有无可指责的人，只作一个妇人的丈夫，儿女也是信主的，没有人告他们是放荡不服约束的，就可以设立。监督既是上帝的管家，必须无可指责，不任性，不暴躁，不因酒滋事，不打人，不贪无义之财；乐意接待远人，好善，庄重，公平，圣洁自持；坚守所教真实的道理，就能将纯正的教训劝化人，又能把争辩的人驳倒了。 \[多1:5b-9\]*

保罗列举的许多长老或监督的条件，这些条件不外乎是要我们选一个 “好人”；不是选一个有才华的人，有能力，受欢迎，有身份，等条件的人，而是要选一个圣洁有见证的人，坚守真理并传讲真理的人。

我们往往把圣洁和真理当作选举长老的条件 \-- 这当然是圣经白纸黑字的条件。但是圣洁和真理到底是长老的全部条件还是前提？因为我们翻开圣经就会发现圣经似乎并没有充足的教导；除了圣洁和真理之外再没有别的条件了。然而光是考虑圣洁和真理，教会中能做长老的似乎有一大票人，导致我们除此之外还要再考虑才华，能力，身份等条件。圣洁和真理成了选举长老的前提，因为这样的人在教会中比比皆是。(是这样吗？还是我们把圣洁和真理看得太肤浅了？)

我们都知道，好人未必是有能的人。当我们到了传承的那一天，或许会焦虑没有有能力可以接棒。如果到了那一天在我们面前的只有无能的好人与有能的坏人，一定要坚持传给好人，不要害怕服事翻船。

## **解决教会面临的问题**

在大方向上，神的仆人要树立典范，坚守并传讲真理，然后设立合适的领袖。然而，各个教会会面临不同的问题，神的仆人也要竭力地解决。例如，保罗写信给加拉太，哥林多，歌罗西，帖撒罗尼迦教会，起因都是这些教会面临了不同的状况与难题。以提多书为例，克里特岛的人似乎在当时不是非常 “文明”，所以保罗在特别嘱咐提多：*这话 (耶稣受死复活真理) 是可信的。我也愿你把这些事切切实实地讲明，使那些已信上帝的人留心做正经事业* (多8a)

**思想 \--** 我们的 (地方) 教会面临什么挑战呢？ 

# **教会管教信徒的目标**

提摩太后书与提多书都提到教会要管教信徒。

*主的仆人不可争竞，只要温温和和地待众人，善于教导，存心忍耐，用温柔劝戒那抵挡的人；或者神给他们悔改的心，可以明白真道 \[提后2:24-25a\]*

*「克里特人常说谎话，乃是恶兽，又馋又懒。」这个见证是真的。所以，你要严严地责备他们，使他们在真道上纯全无疵 \[多1:12b-13\]*

*这些事你要讲明，劝戒人，用各等权柄责备人；不可叫人轻看你。 \[多2:15\]*

教会应该如何管教信徒呢？当然是按照圣经的教训。那么，我们是要按照提摩太后书 “存心忍耐，用温柔劝解” 呢？还是要按照提多书 “严严地责备”，“用各等权柄责备人” 呢？我们可能会说 “责备人会破坏合一，保罗那是给提多的特别指示”。那么，为什么温柔劝解为什么不是给提摩太特别的指示呢？而且，温柔也会导致姑息养奸。

或许是提摩太与提多各有不同的恩赐，所以保罗给他们指示不同。(当然，也有可能是他们需要操练的方向不同... 圣经并没有就此明说。) 但是保罗给他们指出的目标是相同的；就是要信徒们圣洁并且明白真道：“或者神给他们悔改的心，可以明白真道”，“使他们在真道上纯全无疵”。所以教会并不是一定要用温柔或者只能用严厉来管教信徒，而是要定睛在目标，用各种不同的恩赐去达成。

# **腓利门书**

## **背景**

腓利门书是保罗第一次被囚禁时所写的。这封信不是为了解决教会的问题，而是解决两个人之间的纠纷：阿尼西谋与腓利门。

腓利门是歌罗西教会的一位弟兄，甚至歌罗西教会在他家聚会 (门1:2)，而阿尼西谋是腓利门的一个奴隶。阿尼西谋逃离了腓利门，并且不远千里地去到罗马找到当时正被囚禁的保罗。圣经没有交代为什么阿尼西谋逃离腓利门的原因，也没有交代为什么阿尼西谋要去找保罗。(奇怪的是，很有可能阿尼西谋找到保罗的时候，歌罗西教会的以巴弗也在那里为了歌罗西教会遭遇的问题向保罗求教 (门1:23)。) 当时奴隶逃离主人是一件很严重的事。主人可以把逃跑的奴隶处死，或者主人愿意赦免奴隶，就可以用烙铁在奴隶的额头上烙一个 “F” 记号，从此他这一生走到哪里人都知道他是一个不忠心的奴隶。

从腓利门书可以确定的是，阿尼西谋从哥林多逃到罗马并且找到了保罗。在哪里保罗向阿尼西谋传讲了福音，并且阿尼西谋信靠了福音，成了一个重生得救的新人。然后保罗就写了腓利门书，劝腓利门饶恕阿尼西谋，并且因为他已经听信了福音，如今是我们的一位弟兄。这封信就由阿尼西谋亲自带回去。活在现代的我们可能很难想象这对腓利门和阿尼西谋都是很大的挑战：不只是对于奴隶制度的挑战，更是属灵的挑战

我们可以不断地猜测腓利门书背后那没有交代清楚的来龙去脉，但我们跟应该思考的是：为什么神把这封个人恩怨相关的书信放在圣经当中呢？

腓利门书好像是一个多面向的隐喻，把我们信仰的许多面向藉由保罗，阿尼西谋，与腓利门之间的关系表现出来。

## **隐喻**

既然说是隐喻，代表以下从腓利门书而来的领受都**不是圣经白纸黑字的教导**。这些领受对我们的灵命固然有所帮助，但如果有人不赞同这些领受也不需要坚持甚至争论。并且我们必须适度解读隐喻，不要走到钻牛角尖的地步。

**新人的真意[^2] \--** 阿尼西谋听信了福音以后，保罗要他回到腓利门那里去。这对阿尼西谋是一件大事！尽管保罗在信中请求腓利门宽恕阿尼西谋，并且要待他如在主内的弟兄 (他真的是主内的弟兄)，这在当时的奴隶制度之下仍是一件大事。我们信靠耶稣的人是神在基督里新造的人。这不是说我们过往对人的亏欠都与我们不再有关，不是神帮我们把这些都一笔勾销。反之重生得救的新人应有直面过犯，积极悔改，弥补过错的勇气[^3]。

**有益的人[^4] \--** 阿尼西谋这名字的意思是 “有益处”。保罗用一个近似文字游戏的方式劝腓利门赦免阿尼西谋并且要把他看作一位弟兄：

*为我 (保罗) 在捆锁中所生的儿子阿尼西谋求你 (腓利门)。他从前与你没有益处，但如今与你我都有益处。 \[门1:10b-11\]*

我们可以把自己比作阿尼西谋：本来是一个逃走的奴隶，是无用的。但是福音改变了他，如今他成为一个新造的，有用的人。这就像我们在福音里重生一样。

**饶恕的意义 \--** 保罗这样劝腓利门接纳阿尼西谋：

*你 (腓利门) 若以我 (保罗) 为同伴，就收纳他 (阿尼西谋)，如同收纳我一样。他若亏负你，或欠你什么，都归在我的帐上；我必偿还，这是我—保罗亲笔写的。我并不用对你说，连你自己也是亏欠于我。 \[门1:17-19\]*

我们可以把自己比作腓利门，保罗比作耶稣，而阿尼西谋是一位亏欠我们的人。神要我们赦免别人，因为他会主持公义，我们所被亏欠的神会补偿我们；这是神给我们的恩典。这实在是恩典，因为我们自己也是亏欠神的。
